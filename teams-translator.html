Beta
OpenAI Realtime API 세션 두 개를 병렬로 돌려 양방향 동시 번역을 구현한 Windows 데스크톱 앱. 가상 오디오 케이블로 Teams와 연결되어 별도 플러그인 설치가 필요 없습니다.
양방향 동시한↔영 세션 2개 병렬 동작
API 키 보안OS 자격 증명 저장소(keyring)
자동 재연결지수 백오프 + 세션 자동 복구
오디오 엔진24kHz 리샘플링 · WASAPI
주요 기능
내 한국어 → 영어 음성으로 Teams 마이크에 송출
상대 영어 → 한국어 음성으로 내 헤드폰에 재생
네트워크 끊김 시 자동 재연결 & 설정 복구
401·429·네트워크 오류를 한국어로 친절하게 안내
모델·voice 설정 외부화(config.json) & 회전 로그
필요 환경
Windows 10/11 · VB-Audio Virtual Cable · OpenAI API Key
